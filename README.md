# purf-browser
Lightweight browser built in C++ and QTWebkit
# Installing & Building
Void Linux:
```
sudo xbps-install -S qmake make qt5-base-devel qt5-webkit qt5-webkit-devel
```
Arch Linux:
```
sudo pacman -S qmake make qt5-base-devel qt5-webkit qt5-webkit-devel
```
# Cloning & Building
```
git clone https://github.com/purf-qt/purf-browser.git
```
Building:
```
cd purf-browser
qmake browser.pro
make
```
# Launching
```
cd purf-browser
./browser
```
# Enjoy!
